# Clash Royale Sound Assets

Place your Clash Royale sound files in this directory for the Pomodoro timer sound system.

## Required Sound Files

Due to copyright restrictions, sound files are **not included** with Recallify. You must obtain them yourself from legitimate sources.

### File Names (must match exactly):

1. **mega-knight.mp3** - Mega Knight deployment sound ("HUUUUUU MEGA KNIGHT")
   - Plays when: Work session starts

2. **hog-rider.mp3** - Hog Rider battle cry ("HOG RIDERRRR!")
   - Plays when: Short break starts

3. **electro-wizard.mp3** - Electro Wizard deployment ("ELECTRO WIZARDYYYY!")
   - Plays when: Long break starts

4. **king-angry.mp3** - Angry King emote ("GRRRR!")
   - Plays when: 30 seconds before break ends

5. **king-laugh.mp3** - Laughing King emote ("He-He-He-Haw!")
   - Plays when: 1 minute left in Pomodoro

6. **goblin-cry.mp3** - Goblin crying emote
   - Plays when: Session is abandoned

## Technical Requirements

- **Format**: MP3
- **Max file size**: <100 KB per file
- **Duration**: ~1-2 seconds each
- **Audio normalization**: Ensure consistent volume across all files

## How to Obtain Sounds

1. Extract from the Clash Royale game files (if permitted by Supercell's terms)
2. Record from gameplay (ensure compliance with Supercell's content policy)
3. Use publicly available sound effect libraries with proper licensing

## Testing Sounds

After placing the files:
1. Open Recallify
2. Go to Settings > Pomodoro
3. Scroll to "Clash Royale Sounds"
4. Enable the feature
5. Use "Test Sound" buttons to verify each audio file

## Troubleshooting

**Sounds not playing?**
- Verify file names match exactly (lowercase, with hyphens)
- Check file format is MP3
- Ensure files are in this directory: `public/sounds/clash-royale/`
- Try playing files directly to verify they're not corrupted
- Check browser console for error messages

**Volume too loud/quiet?**
- Adjust the volume slider in Settings > Pomodoro
- Normalize audio levels using audio editing software
